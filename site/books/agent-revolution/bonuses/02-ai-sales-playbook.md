# Bonus #2: AI Sales Playbook

**Retail Value: $29.95**

A practical sales operating system for solo founders and small teams using AI to generate pipeline, qualify leads, and close faster without sounding robotic.

## What’s Inside

### 1) AI Offer Positioning Sprint
- 15-minute positioning worksheet
- ICP clarity prompts (industry, role, pain, trigger events)
- Fast “before/after” transformation messaging template

### 2) Outreach Engine (Cold + Warm)
- 20 cold outbound scripts (email/DM)
- 10 warm intro scripts
- Follow-up ladder (Day 2, 4, 7, 14)
- Personalization prompt framework that avoids generic AI copy

### 3) Discovery & Qualification Framework
- AI-assisted discovery call prep checklist
- Question bank for pain, urgency, authority, budget, timing
- Qualification scoring template (Hot / Warm / Nurture)
- Call summary + next-step recap prompt

### 4) Proposal & Objection Handling Pack
- Proposal structure templates (short-form + full)
- Objection response matrix (price, timing, trust, fit)
- “Risk reversal” language blocks
- Close scripts for soft-close and direct-close scenarios

### 5) Follow-Up + Pipeline Management
- Weekly pipeline review prompt
- Deal rescue templates for stalled opportunities
- Re-engagement campaign scripts
- CRM note standard and stage progression checklist

### 6) KPI Dashboard Starter
- Lead-to-meeting conversion tracker
- Meeting-to-proposal tracker
- Proposal-to-close tracker
- Weekly revenue forecast template

## Recommended Workflow
1. Define offer + ICP with the Positioning Sprint.
2. Launch outbound scripts in one channel first.
3. Run discovery with qualification scoring.
4. Send proposal within 24 hours using templates.
5. Execute follow-up ladder until close or nurture.
6. Review pipeline every Friday and refine messaging.

## Implementation Notes
- Best results come from using your own voice in first drafts, then refining with AI.
- Keep personalization factual (company, role, trigger, relevant pain).
- Track conversion rates by script variant; kill what underperforms quickly.
