FAQ

Common questions

Do I need to be a developer to use this book? +
Not a professional developer, no. If you're comfortable using a terminal, editing config files, and following step-by-step instructions, you have enough skill. The book assumes comfort with basic command-line operations: not software engineering expertise.
What hardware do I need? +
A Mac Mini, a Linux server, or even a $5/month VPS. The gateway is lightweight: it's a Node.js application that connects to AI APIs. You don't need a GPU or expensive hardware. The book covers all setup options in detail.
Which AI models does OpenClaw support? +
All of them. OpenClaw works with OpenAI (GPT-4, GPT-5), Anthropic (Claude), Google (Gemini), DeepSeek, and local models via Ollama. You can switch models anytime or use different models for different tasks.
Is OpenClaw really free? +
Yes. OpenClaw is MIT-licensed, fully open source. You still need to pay for AI model access (API costs or subscriptions like Claude Max / ChatGPT Plus), but the gateway software itself is completely free.
